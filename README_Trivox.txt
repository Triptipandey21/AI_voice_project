
# Trivox

**Trivox** is a multi-platform trivia quiz app with real-time scoring and leaderboard. It is built to provide an engaging and responsive quiz experience using modern web technologies.

## 🚀 Features

- Interactive trivia quizzes
- Real-time scoring and leaderboard
- Multi-device support (Responsive UI)
- Admin panel for question management
- Easy to deploy and customize

## 🛠 Tech Stack

- Frontend: HTML, CSS, JavaScript (or React)
- Backend: Node.js, Express
- Database: MongoDB
- Deployment: Vercel or Render

## 📦 Installation

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/trivox.git
   cd trivox
   ```

2. Install dependencies:
   ```
   npm install
   ```

3. Start the development server:
   ```
   npm start
   ```

4. Navigate to `http://localhost:3000` to view the app.
